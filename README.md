# Каталог образовательных программ в творческой сфере Москвы

Интерактивный каталог для взрослых, желающих обучаться в Москве по направлениям:
- театральное искусство
- киноискусство
- театральная импровизация
- музыкальные представления
- психотерапевтический театр

## Как обновлять данные

Для обновления каталога отредактируйте файл `index.html`, следуя структуре таблиц. 
Каждая из 4 секций должна содержать не менее 15 организаций с полным перечнем программ.

## Источники данных

Данные взяты из публичных источников 2020-2024 гг.:
- Реестр лицензий Рособрнадзора
- Официальные сайты организаций
- Платформы "Учеба.ру", "Курсов.рф", "Афиша"
